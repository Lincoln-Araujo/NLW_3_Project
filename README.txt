NextLevelWeek 3 Edição
Participei de uma semana de imersão em HTML5, CSS3, Javascript e NODE JS para construirmos juntos uma aplicação voltada a Abrigos e Casas que cuidam de jovens em situação de Abandono. A ideia é localizar tais instituições em uma determinada região e divulgar algumas informações sobre elas, inclusive para visitação.

Passamos deste o Front End, Back End a construção de banco de dados para receber as informações dadas para quem quiser indicar uma nova instituição.

No Front End, focamos bastante na codificação de uma página responsiva onde poderia ser lida de qualquer tipo de aparelho sem perda de conteúdo ou desconfigurando a aplicação e que também fosse capaz de colher informações para um formulário.

Foi a primeira vez que tive contato com Back End e nessa parte fizemos toda a configuração base para trazer o front end para essa etapa, incluisive aqui, criamos um servidor utilizando NODE JS.
